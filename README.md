# Basic CNN Image Classification (CIFAR-10)

## 📌 Project Overview

This project implements a **Convolutional Neural Network (CNN)** to classify images from the **CIFAR-10 dataset** using TensorFlow and Keras.

The model learns to recognize objects such as airplanes, cars, cats, and dogs from small images.

---

## 🧠 Concepts Used

* Convolutional Neural Networks (CNN)
* Image Classification
* Feature Extraction
* MaxPooling
* Flatten Layer
* Dense Layers
* Model Training & Evaluation

---

## 📊 Dataset

The project uses the **CIFAR-10 dataset**, which contains:

* 60,000 images
* 10 different classes
* Image size: 32 × 32 pixels

Classes include:

* Airplane
* Automobile
* Bird
* Cat
* Deer
* Dog
* Frog
* Horse
* Ship
* Truck

---

## ⚙️ Model Architecture

The CNN model consists of the following layers:

Conv2D → MaxPooling → Conv2D → MaxPooling → Flatten → Dense → Output
